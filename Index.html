The Ambiguous Images dataset is a labeled dataset of 460 images from various countries and cultures.

The Ambiguity of the images are defined based on the following measures of cultural Ambiguity:

Complexity (C) - Number of identifiable cultural elements

Diversity (D) - Number of unique cultures attributable to elements in the image

Geographical/Cultural Distance (G) - How far are the countries geographically ( Proportional measurement 0-1)

Rarity (R) - Population size of a cultural group

Historical Value (H) - Resembelance with modern culture
